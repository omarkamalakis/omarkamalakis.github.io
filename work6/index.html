Homelab Part 2: Installing the Operating System

Gabe Kamalakis published on January 08, 2020
2 min, 294 words

After verifying that our host machine works, I install ProxMox VE onto the server on a whim...

ProxMox - a whim:

The decision to use ProxMox as the Host OS was made in around 5 minutes after the PC wouldn't read my fedora installation file which I had burned in using Etcher. I wiped the USB stick and wrote PVE onto it using DD instead (which was around as easy as Etcher and much faster). There are many benefits to using ProxMox, namely the web monitoring and configuration, the ease of installation, and how good the performance is. Because ProxMox uses both KVM and LXC containers, performance is much better than something like VirtualBox. Proxmox also makes using these very easy with their web GUI.

I also noticed that it's possible (and relatively easy) to add more computers as nodes to create a Beowulf Cluster. I don't really have a need to do any HPC work, but this is a very exciting possibility.

Installation was incredibly easy, simply plugging in the USB and booting to it was very straightforward and it installed (fully configured) in around 5-10 minutes. I opted to just use an EXT4 filesystem since I wasn't familiar with ZFS or BTRFS quite yet, though that may come in the future. After installing and reserving the IP address on the router, I logged into the web server to be greeted by this.

All that's left is to start building virtual machines and containers.
